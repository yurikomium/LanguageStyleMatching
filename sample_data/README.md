# Sample Data

このディレクトリには、Language Style Matching (LSM) アルゴリズムのデモンストレーション用のサンプル対話データが含まれています。

## データ形式

各CSVファイルには以下の列が含まれています：
- `speaker`: 発話者 ("female" または "male")
- `text`: 発話テキスト（日本語）
- `start`: 発話開始時刻（秒単位）

## ファイル命名規則

ファイル名は以下の形式に従います：
```
{セッションID}_{ラウンド}_{グループ}_A_F{女性ID}_M{男性ID}_concat.csv
```

例：
- `01_1_1_A_F001_M001_concat.csv`: セッション01、ラウンド1（初回）、女性001と男性001の対話
- `01_2_1_A_F001_M001_concat.csv`: セッション01、ラウンド2（2回目）、同じペアの対話
- `02_1_1_A_F002_M002_concat.csv`: セッション02、ラウンド1、女性002と男性002の対話

## 注意事項

これらのデータは架空のデモンストレーション用データです。実際の対話データではありません。
LSMアルゴリズムの動作確認と実装の検証を目的としています。
